Software interests & skills

I have had the opportunity to work with various people including classmates, co-workers and university professors on many software projects and I am always looking forward to working with them or others again. I am mainly interested in developing software either as commercial products or software development tools, such as those listed here.

Application Development

Overview: I love developing applications, specifically for Android, web and Windows environments. My preferred approach is to create distributed back-ends using web services based on either dedicated or cloud servers and use client versions on Android, Web or Windows which interface the services using asynchronous calls.

Android

I love developing modern Android applications which utilize the Material design specifications. So far, I have implemented several projects including my BSc thesis aMazeChallenge, a currency converter, a prototype chat application and collaborated to create the client version of a mobile distributed ERP system.

Java, XML, Android SDK, MVC, Material design, Maven etc

Web

I have worked as a part-time web developer in the past and created a variety of websites using a custom Wordpress-like CMS. I also tested and debugged a lot of exiting websites. Apart from work, I have also worked on several freelance and personal projects including promotional websites, a prototype chat application, my personal website nkasenides.com (You are here!), the Code Cyprus Treasure Hunt Web App, the Cyprus Water app and more.

HTML5, CSS3, JavaScript, JSON, JQuery, Bootstrap, MaterializeCSS, Wordpress

Windows

I also like to get old-school sometimes and create classic desktop applications for the Windows environment using the .NET framework. Some examples include an unbeatable Tic Tac Toe game and a Windows client version of an old personal project called Dreammate which was used as a code generator.

C++, C#, XAML, ASP.NET, Windows 10 SDK

Server-side

My prefered approach to "power" applications is to create and maintain distributed back-ends that utilize web services to provide access to data and functionality at the server side. I also like using server-less architectures, such as Google's Firebase with cloud functions. In the past, I have maintained server-side scripts for work projects and created a variety of web-based APIs.

PHP, Java (mostly servlets), MySQL, MSSQL, Apache, Google App Engine & Firebase

Software tools

I typically use Java-XML, HTML-CSS-JavaScript and C#-XAML to develop Android, web and Windows applications respectively. My IDEs of choice are Android Studio, JetBrains WebStorm and Microsoft Visual Studio. I also actively use or have used the following tools in the past: IntelliJ IDEA, Eclipse, NetBeans, PhpStorm, Dreamweaver, phpMyAdmin, DataGrip, Navicat, MSSQL Server, GitHub, GitLab (both with Git shell) and more.

Further interests

I am also interested in developing software development tools such as ORMs, code generators and more. I am currently working on a code generator library for various programming languages and frameworks covering the majority of the software development stack (starting from entity-database mapping and ending at UI generation). I also have some experience creating parsers and compilers for custom markup/programming languages and using the ANTLR4 library. To test some of my skills, I created a very small language with its own compiler, called Ray.

Other experiences

I will usually work with just about anything if properly motivated (ok, maybe not Python). I have also programmed in Pascal and Python in the past (although I hate the latter). Some additional recreational activities include experimentation with the Raspberry Pi 2 using GPS, cameras and touchscreens. I have also experimented with Arduino in the past and used motors, LEDs and other hardware to create a robot snowman. The opportunities are endless!
