aMazeChallenge

aMazeChallenge is a code-based multiplayer game based on Android powered by Google's App Engine which teaches programming to players by allowing them to control a bot by writing basic code in Blockly, uploading it and then competing against each other live to solve a maze. Created in collaboration with my BSc final year project supervisor, Dr. Nearchos Paspallis.

SourceFusion

SourceFusion is an in-development project which generates code for use in distributed applications and for almost the entire development stack (starting from entity-database mapping and ending at UI generation). This code-generation library allows software engineers to abstractly define various entities (in a programming-language agnostic way) and then generate code for classes, database/store items, web services, client requests, UI and more in a set of selected programming languages or frameworks.
