SourceFusion

SourceFusion is an in-development code-generation library which allows software engineers to abstractly define various entities (in a programming-language agnostic way) and then generate corresponding code in a set of specified programming languages or frameworks. The tool is aimed for use in distributed applications with mainly thin clients which are powered by web-based APIs. It features its own authentication framework and allows the generation of:

  • Entity classes (in C++, C#, Java, JavaScript, PHP).
  • Database helpers (for MySQL w/ Java, MySQL w/ PHP, Google Datastore w/ Java).
  • Web services (as Java Servlets or PHP endpoints).
  • Forms (as HTML or Android forms).
  • Documentation (in HTML).

SourceFusion aims to relieve the software engineer from various development steps such as the authentication process, the creation of CRUD-style web services for each entity, web service documentation, form creation, client-side response processing and more, with the ultimate objective being to save effort and time that would be spent doing repetitive work in each project.
